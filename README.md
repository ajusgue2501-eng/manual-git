## Comandos git

>[!NOTE]
>
>Nos son algunos de los comandos mas utilizados de git


| Comando | Acción
|---|---|
| `git init` | Inicializa un repositorio local.
| `git clone` | Clona un repositorio remoto.
| `git add` | Añade archivos al área de preparación.
| `git commit` | Guarda los cambios en el historial.
| `git log` | Muestra el historial de commits.
| `git checkout` | Cambia entre ramas o versiones.
| `git branch` | Crea o lista ramas.
| `git push` | Envía los cambios al repositorio remoto.
| `git pull` | Descarga y fusiona cambios del remoto.
| `git merge` | Fusiona ramas.
